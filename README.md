# Sistema de Controle de Internações Hospitalares

## Sobre o Projeto

Este projeto foi desenvolvido para a disciplina de Programação Orientada a Objetos utilizando Java, JDBC e Banco de Dados Relacional. O sistema tem como objetivo realizar o gerenciamento de médicos, pacientes e internações hospitalares, aplicando conceitos de arquitetura em camadas e padrões de projeto.
